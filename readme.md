# CNF Parser
with CYK algorithm

## Getting Started

These instructions will tell you how to compile and play the game.

### Compiling

To compile, in folder cnf-parser execute

```
$ make
```

## Running the game

To run the program, simply

```
$ ./main
```
and enjoy the game.
